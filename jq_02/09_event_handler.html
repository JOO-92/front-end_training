이벤트 등록 메소드

이벤트란 웹문서에서 사용자가 취하는 모든 행위를 가리킨다. 그리고 이벤트 핸들러는 이벤트가 발생했을 때 실행되는 일련의 실행 코드를 가리킨다. 이벤트 등록 메소드를 이용하면 사용자가 지정한 요소에서 어떠한 특정 동작이 일어났을 때 지정된 실행문을 실행시킬 수 있다.

이벤트 등록 메소드에는 하나의 이벤트만 등록할 수 있는 이벤트 등록 메소드와, 하나 이상 등록할 수 있는 그룹 이벤트 등록 메소드가 있다.

$('button').hover(function(e){
  // hover는 커서가 닿을때와 떠날때로 구분된다.
  if(e.type == "mouseenter"){
    $(this).css("color", "cyan");
    $(this).css("backgroundColor", "gray");
  }else{
    $(this).css("color", "tomato");
    $(this).css("backgroundColor", "teal");
  }
})

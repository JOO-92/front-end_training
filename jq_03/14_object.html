객체

객체란 여러 종류의 속성(property)을 가지고 있는 데이터의 집합이다. 각 속성에는 데이터가 저장되어 있으므로 '각종 데이터를 하나로 모아서 하나의 변수로 처리할 수 있는 데이터'라고 할수도 있다. 언뜻 보면 배열과 비슷하게 느껴질 수도 있으나, 그 성격이 서로 조금씩 다르다.

객체 작성 방법

객체를 작성할 때는 중괄호{}를 사용한다. 배열과 마찬가지로 생성한 객체를 변수에 저장하는 것이 일반적이므로 보통은 다음과 같은 형식을 사용한다.

1
2
3
4
5
const 변수명 = {
  속성: 데이터,
  속성: 데이터,
  속성: 데이터
};

아래는 직접 객체를 작성한 예이다.

1
2
3
4
5
const person = {
  age: 16,
  job: "student",
  town: "seoul",
};

속성의 개수에는 제한이 없다. 각 프로퍼티에 접근하는 방법은 아래와 같다.

1
2
3
console.log(person.age); 
person.town = "bucheon"; // 속성값의 변경
console.log(person.town)

객체 사용법을 알아야 하는 이유

추후 서버로부터 응답을 받아왔을 때 객체형의 데이터를 처리해야 하는 상황이 생길 수 있다.

for문

for문 또한 while과 같은 반복문이다. for문의 형태는 while에 비해 좀 더 직관적인 느낌이 있어 비교적 더 자주 사용된다. for문의 기본형은 다음과 같다.

1
2
3
for(초기값; 조건식; 증감식){
  실행문;
}

for문 예제

for문을 사용해 1부터 10의 총합을 구하는 예제이다.

1
2
3
4
var total = 0;
for(var i = 0; i <= 10; i++){
  total += i;
}

  if(i % 2 == 1){
    document.write('<span style="color:red;">' + i + '</span><br>');
  } else {
    document.write('<span style="color:blue;">' + i + '</span><br>');
  }
} 

for문의 중첩

for문 안에 for문을 사용한 것을 중첩 for문이라고 한다.
이해를 돕기 위한 표현 : 가령 for문 (1)안에 for문 (2)가 들어있는 경우에, (1)은 행이고 (2)는 열이라고 생각할 수 있다.
아래 코드는 for문의 중첩을 활용해 구구단을 출력하는 예제이다.

1
2
3
4
5
6
for(var dan = 2; dan <= 9; dan++){
  for(var gob = 1; gob <= 9; gob++){
    document.write(dan * gob, ' ');
  }
  document.write('<br>');
}

배열 객체

배열

앞서 변수에는 데이터가 한 개만 저장된다고 하였지만, 여러 개의 데이터를 하나의 변수에 저장하는 방법도 존재한다.
여러 개의 데이터를 하나의 저장소에 저장하려면 배열(Array) 객체를 생성하면 된다. 배열객체를 생성하는 기본형은 다음과 같다.

1
2
var 배열_변수명 = [값1, 값2, 값3, ...];
var 배열_변수명 = new Array(값1, 값2, 값3, ...);

배열 객체 저장된 데이터 불러오기

배열 객체에 저장된 데이터를 불러올 때의 기본형은 다음과 같다.

1
배열_변수명[인덱스번호];

다음은 객체에 저장된 값들을 모두 출력한 예이다.

1
2
3
4
5
6
var arr = [1,2,3,4,5];
document.write(arr[0] + '<br>');
document.write(arr[1] + '<br>');
document.write(arr[2] + '<br>');
document.write(arr[3] + '<br>');
document.write(arr[4] + '<br>');

배열의 데이터에 접근할 때 주의해야 할 점은, 첫번째 인덱스가 0으로 시작한다는 점이다.

배열의 유용한 메소드 및 속성

배열 역시 다른 객체들처러 다양한 메소드를 가지고 있다.

종류 의미
reverse() 배열 객체의 데이터 순서를 거꾸로 바꾼 후 반환한다.
sort() 배열 객체의 데이터를 오름차순으로 정렬한다.
pop() 배열에 저장된 데이터 중 마지막 것을 삭제한다.
push(data) 배열 객체 마지막에 데이터를 삽입한다.
length 배열 객체의 데이터 개수를 구한다.
splice(index1, n, data) index1부터 n개만큼 데이터를 지우고 data로 대체한다.

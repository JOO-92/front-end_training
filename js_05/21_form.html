폼의 입력 내용 가져오기

이벤트 테스트

먼저, 다음과 같은 HTML 코드를 추가한 다음 form 내부의 버튼을 클릭해본다.

1
2
3
<form action="#" id="form">
  <input type="submit" value="검색">
</form>

그런 다음 이벤트 설정을 위한 프로그램을 작성한다(자바스크립트).

1
2
3
document.getElementById('form').onsubmit = function(){
  console.log('클릭했습니다.')
}

이벤트

링크 또는 버튼을 클릭했을 때, 키보드를 조작했을 때, 또는 페이지 읽기가 끝났을 때나 다른 ㅍ이지로 이동하기 직전 등 다양한 시점에 브라우저의 이벤트가 발생한다. 위 코드에서 사용된 onsubmit도 이벤트의 한 종류이다.
function(){~} 은 '함수'라고 불리는 것으로, 이벤트 발생 시의 동작을 {~}의 ~ 부분에 작성하는 것이 일반적이다.

1
2
3
4
//이벤트 발생 시 처리할 내용을 정의하는 일반적 형태
document.getElementById(요소_아이디).이벤트종류 = function(){
  처리할 내용
}

입력 내용 읽어서 가져오기

사용자가 '검색' 버튼을 클릭한 시점에 텍스트 필드에 입력되어 있는 내용을 읽도록 해보자.
먼저 아래와 같이 HTML 코드를 수정한다.

1
2
3
4
5
<form action="#" id="form">
  <input type="text" name="search">
  <input type="submit" value="검색">
</form>
<p id="output"></p>

name 속성 값은 입력한 데이터가 제출(submit)될 때 데이터의 이름으로 사용된다. 따라서 폼 입력값의 처리를 위해서는 매우 중요한 속성이 이 name이다.

그렇다면 이제 전송 버튼을 클릭한 시점에 name 속성을 사용해 내용을 읽고 그것을 p 태그에 출력하도록 자바스크립트 코드를 작성해보겠다.

1
2
3
4
5
document.getElementById('form').onsubmit = function(){
  var search = document.getElementById('form').search.value;
  document.getElementById('output').textContent = search;
  return false;
}

함수

함수란?

변수에는 데이터만 저장할 수 있고, 실행문을 저장할 수는 없다. 하지만 함수를 사용하면 실행문을 메모리에 저장했다가 필요할 때마다 호출하여 사용할 수 있다.
함수를 정의하는 기본 형태는 다음과 같다.

1
2
3
function 함수이름(){
    // 실행문
}

다음과 같은 형태 또한 가능하다.

1
2
3
변수명 = function(){  
    // 실행문
}

function changeBgColor(){
  var colors = ['red', 'yellow', 'green', 'blue'];
  var randNum = Math.floor(Math.random() * 4);
  document.getElementsByTagName('button')[0].style.backgroundColor = colors[randNum];
}

매개변수

매개변수란 함수 실행 시에 함수에게 전달할 값을 지정할 수 있는 변수를 의미한다. 매개변수를 통해 전달된 값은 함수 내 실행문에서 사용될 수 있다.

1
2
3
4
5
6
7
// 매개변수의 개수는 특별히 제한되어 있지 않다.  
function 함수명(매개변수1, 매개변수2){
    // 실행문
} 
 
//호출 시에는 다음과 같이 호출한다
함수명(전달값1, 전달값2);

함수와 return

return의 역할

return은 함수에서 결과값을 되돌려 줄 때 사용된다. 그리고 함수에서 return문이 실행되면 실행문이 강제로 종료되기도 한다.

  return ++value;

function checkSomething(value){
  if(value % 2 == 1){
    return "홀수입니다."
  }else{
    //과연 결과는?
    return
  }
}

document.write(checkSomething(3) + "<br>");

재귀 함수 호출(약간 어려움 주의)

함수 실행문 내에서 함수를 다시 호출하는 것을 재귀(recursion)이라고 한다. 재귀 함수 호출의 기본형은 다음과 같다.

  실행문;
  함수이름(); //함수 안에서 같은 이름의 함수를 호출한다. 
}

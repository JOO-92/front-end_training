querySelectorAll 메소드

querySelectorAll 메소드는 () 안에 지정한 CSS 선택자와 일치하는 모든 요소를 가져오는 메소드이다.

1
2
// 일치하는 모든 요소를 가져온다. 
document.querySelectorAll('CSS 선택자');

data-* 속성과 dataset 속성

data-* 속성은 * 부분을 자유롭게(영소문자) 정할 수 있는 속성이다. 이렇게 지정한 속성은 자바스크립트 코드를 통해 다음과 같이 읽어들일 수 있다.

최신 자바스크립트

개요

자바스크립트 주기적으로 새로운 버전 및 문법을 출시하고 있다. 또한 노드도 주기적으로 버전을 올리며 변경된 자바스크립트 문법을 반영하고 있다. 따라서 앞으로는 새로운 자바스크립트 문법에 익숙해질 필요가 있다. 여기서는 그 중에서도 실제로 많이 사용되는 몇 가지를 소개한다.

디폴트 파라미터

C++나 파이썬 같은 언어에서는 함수의 인자로 디폴트 값을 선언할 수 있다. 최신 자바스크립트에도 이것이 추가되었다. 따라서 함수를 호출하면서 값을 지정하지 않으면 디폴트 값이 사용된다.

1
2
3
4
function logActivity(name="강윤호", activity="달리기"){
  console.log(`${name}는 ${activity}를 좋아한다.`)
}
logActivity();

파라미터에 인자를 전달하면 새로운 값으로 함수 기능을 수행한다.

1
2
3
4
5
const person = {
  name : "메시", 
  activity : "축구"
}
logActivity(person.name, person.activity);

화살표 함수

화살표 함수를 사용하면 function 키워드가 없어도 함수를 만들 수 있다. 또한 return을 사용하지 않아도 자동으로 값을 반환할 수 있다.

전통적인 방식의 함수

1
2
3
4
var heFrom = function(town){
  return `${town} 출신의 강윤호`
}
console.log(heFrom("대전"));

화살표 함수

1
2
const heFrom = (town) => `${town} 출신의 강윤호`
console.log(heFrom("대전"));
